AST ChangesA list of modifications made which differentiate the AST from the standardized syntax:- 'FieldDecl.java' contains an 'int', "offset"- 'Identifier.java' contains a 'String', "name" as well as a 'Declaration', "declaration"- 'LocalDecl.java' contains an 'int', "offset"- 'MethodDecl.java' contains an 'int', "address"- 'Reference' contains a 'Declaration', "declaration"
